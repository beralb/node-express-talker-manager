<h1 align="center">Nodejs-Express / Project Talker-Manager</h1>
<h3 align="center">This was a project developed in the back-end module of Web Development Course of <a href=https://www.betrybe.com>Trybe</a></h3>

</br>

- Understand what a route is
- Perform asynchronous operations with callbacks and promises
- Read and write scripts that create and consume promises
- Consciously make function calls
- Understand what HTTP is, what an API is, and how the two relate to Express
- Understand what an Express application structure is and how to organize the code
- Create routes and apply middleware
- Write APIs using Node and Express
</br>
